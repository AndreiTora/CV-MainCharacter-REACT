# CV - Main Character (React)

CV estilo videojuego hecho con React + Vite.

## Demo

https://andreitora.github.io/CV-MainCharacter-REACT/

## Tecnologías

- React
- Vite
- Sass
- Framer Motion
- gh-pages

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo local

```bash
npm run dev
```

## Build de producción

```bash
npm run build
```

## Publicar en GitHub Pages

```bash
npm run deploy
```

Este comando:

1. Ejecuta el build (predeploy).
2. Publica la carpeta dist en la rama gh-pages.

## Estructura principal

```text
src/
  components/
  data/profile.json
  App.jsx
  index.scss
```

## Personalización rápida

- Edita los datos personales en `src/data/profile.json`.
- Ajusta estilos globales y responsive en `src/index.scss`.
- Modifica la estructura visual en `src/App.jsx` y componentes de `src/components/`.
